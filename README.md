# Kenyan Dashboard – AI-Powered Learning Platform

A responsive, interactive dashboard built for Kenyan learners that tracks progress, lessons, and achievements. Powered by **Supabase**, **Vercel**, and **React**, with real-time AI tutoring integration.

## Features

- **AI Tutor Integration** – Real-time conversational AI to guide learning.
- **Progress Tracking** – XP, day streak, accuracy, and weekly progress charts.
- **Badges & Gamification** – Reward system to motivate consistent learning.
- **Recent Lessons** – Quick access to last lessons with performance stats.
- **Edge Functions** – Custom serverless functions hosted on Supabase.
- **Responsive Design** – Works on desktop, tablet, and mobile.

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS, Framer Motion
- **Backend:** Supabase (Database + Edge Functions)
- **Deployment:** Vercel
- **Icons & UI Components:** Lucide-React, ShadCN/UI
